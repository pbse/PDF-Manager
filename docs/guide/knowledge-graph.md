# Knowledge Graph

Pinnacle doesn't just store documents; it understands the relationships between them.

## The Problem

When you have hundreds of PDFs—research papers, legal cases, or financial records—finding connections is hard. You might remember an organization mentioned in a contract, but which other documents mention it?

## The Solution: Semantic Connections

Pinnacle automatically builds a **Knowledge Graph** as you import and read documents.

- **Entity Extraction:** Pinnacle scans your documents for key entities like Organizations, Dates, People, and Amounts.
- **Visual Mapping:** You can open the **Knowledge Graph** pane to see a beautiful, interactive D3.js visualization.
- **Interactive Nodes:**
    - **Blue Nodes:** Represent Documents.
    - **Red/Green/Yellow Nodes:** Represent Entities (Organizations, Dates, etc.).
- **Click to Discover:** Clicking an entity instantly triggers a cross-document search, showing you everywhere that entity is mentioned across your entire library.

This transforms your static file folder into a dynamic, interconnected intelligence base.
